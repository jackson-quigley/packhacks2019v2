How It Works:

BlockStack:

Our website uses an implementation of the Blockstack which is a platform for decentralized apps.


GutenBlock allows for a decentralized, transaction tracked record of changes on the blockchain. This allows for documents to be not only stored but all changes to be tracked and publicly verifiable


Why?

In a time of malicious hackers, not only is it important to have backups of important historical documents and leaked information, but it is also important to ensure that the data on file has not been changed. Whether it is the government or the site provider, we must keep data secure from changes.


How?

Our program works by sotoring the hashed version of all data in a custom database, Gaia. This database is build upon the Atlas network, an adapter from the blockchain network. This network allows users to check and verify that a webpage is the same as when it was last checked.


Our program is able to maintain the accuracy through comparing the hashes. If these hashes are different in any way, then the webpage has been changed.
